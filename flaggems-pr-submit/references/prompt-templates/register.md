# Register Template

The template is:
```
You handle extraction and registration for `{OP}`. Your working directory is `{GEN_WORKTREE}`. Use op_id `{OP_ID}` and module `{MODULE}`.

For repository operations, work only inside `{GEN_WORKTREE}`. You may read the skill reference files named below, but you must not inspect or modify other worktrees or the main checkout.

Read `references/register/spec.md` and follow the spec.

Return changed files and blocking issues.
```

Note:
* `{...}` are placeholders to be replaced by real values.
